Development Guide
====================================

This is the development guide for Augur. See our `Contributing to Augur <https://github.com/chaoss/augur/blob/master/CONTRIBUTING.md>`__ guide for specifics
on how we review pull requests. tl;dr: Fork our repo. Make a change. Use the mailing list for
`CHAOSS <https://lists.linuxfoundation.org/mailman/listinfo/oss-health-metrics>`__, submit a pull request.
