Augur Python Library¶

This documentation is for using Augur as a Python library.

Data Source Modules¶

ghtorrent.GHTorrent Uses GHTorrent and other GitHub data sources and returns dataframes with interesting GitHub indicators
githubapi.GitHubAPI GitHubAPI is a class for getting metrics from the GitHub API
facade.Facade Queries Facade
git.Git Analyzes Git repos directly using dulwich
librariesio.LibrariesIO Handles interaction with https://libraries.io/api to get dependency data
localcsv.LocalCSV
downloads.Downloads Class for retrieveing download information using APIs and web scrapers

Application Modules¶

application.Application Initalizes all classes form Augur using a config file or environment variables
augurplugin.AugurPlugin Defines a base class for Augur plugins to implement
server.Server
runtime Runs Augur with Gunicorn when called
util Provides shared functions that do not fit in a class of their own
