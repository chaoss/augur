Augur Python Library¶

This documentation is for using Augur as a Python library.

Data Source Modules¶

ghtorrent.ghtorrent.GHTorrent

Uses the GHTorrent database to return dataframes with interesting GitHub indicators

githubapi.githubapi.GitHubAPI

GitHubAPI is a class for getting metrics from the GitHub API

facade.facade.Facade

Queries Facade

librariesio.librariesio.LibrariesIO

Handles interaction with https://libraries.io/api to get dependency data

localcsv.localcsv.LocalCSV

Reads local CSV files

downloads.downloads.Downloads

Class for retrieving download information using APIs and web scrapers

Application Modules¶

application.Application

Initalizes all classes from Augur using a config file or environment variables

augurplugin.AugurPlugin

Defines a base class for Augur plugins to implement

server.Server

Defines Augur’s server’s behavior

runtime

Runs Augur with Gunicorn when called

util

Provides shared functions that do not fit in a class of their own
