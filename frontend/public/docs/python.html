Augur Python Library¶

This documentation is for using Augur as a Python library.

Data Source Modules¶

ghtorrent Data source that uses the GHTorrent relational database of GitHub activity.
githubapi Data source that uses the GitHub API
facade Data source that uses Facade’s tables
git Analyzes Git repos directly using git
librariesio Data source that uses the LibrariesIO dependency data
localcsv Loads small included datasets
downloads Data source that gathers download stats from package managers

Application Modules¶

application Handles global context, I/O, and configuration
augurplugin Provides a class that can be used to extend Augur
server Creates a WSGI server that serves the Augur REST API
runtime Runs Augur with Gunicorn when called
util Provides shared functions that do not fit in a class of their own
