-----------------
Controlling Augur
-----------------

Augur provides a variety of ways to interface with its systems.

To control the backend, add repos and repo groups to your database for collection, or perform a variety of utilites, checkout the ``augur`` commands section.

To install the project, start the frontend & backend servers simultaneously, build the docs or rebuild the binaries, checkout the ``make`` commands section.
